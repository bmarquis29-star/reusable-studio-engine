# Reusable Studio Engine
A modular Vanilla JS Canvas starter for rapid creative coding.

## Local Setup
1. Open folder in VS Code.
2. Click "Go Live" using the Live Server extension.

## Deployment
Deployed via GitHub Pages (Settings > Pages > Main Branch).

## Project Ritual
1. Clone Template.
2. Define Signal in `SYSTEM_CHARTER.md`.
3. Map input in `input.js` and render behavior in `loop.js`.
